# 🎮 Cosmic Quest

A retro-style terminal RPG mini game where you explore a mysterious dungeon, battle creatures, and collect treasure.

## Features
- ✨ Turn-based combat system
- 🗺️ Procedurally generated dungeon rooms
- 💎 Treasure and loot collection
- 👹 Encounters with quirky creatures
- 🎯 Multiple endings based on your choices

## How to Play

```bash
python cosmic_quest.py
```

### Controls
- **e** - Explore the room
- **f** - Fight the creature
- **r** - Run away
- **q** - Quit game

## Game Loop
Navigate through the dungeon, decide whether to fight creatures or flee, collect treasure, and survive until you reach the exit!
